# Conversor de Bases 
Converte um número de qualquer base para qualquer outra base entre 2 e 35.

**Aceita apenas entradas válidas. 
Mais instruções no arquivo "main.cpp"**

---
Exemplo: 
>Entrada:  int BASE_ENTRADA [2,...,35]

>	string VALOR

>	int BASE_SAIDA; [2,...,35]

>Output: string SAIDA;
